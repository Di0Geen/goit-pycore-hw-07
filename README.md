# goit-pycore-hw-07

У цьому завданні реалізовано консольного асистента для керування адресною книгою контактів.

Проєкт створено на основі попередньої домашньої роботи та доповнено функціоналом роботи з днями народження.

# Основна функціональність

Асистент дозволяє:

- додавати нові контакти;
- додавати декілька телефонів до одного контакту;
- переглядати всі контакти;
- шукати телефони контакту за іменем;
- редагувати номер телефону;
- додавати день народження до контакту;
- переглядати день народження контакту;
- показувати список днів народження на найближчий тиждень;
- перевіряти правильність введення номера телефону;
- перевіряти правильність формату дати народження;
- завершувати роботу програми командами `close` або `exit`.

# Валідація даних

У проєкті реалізовано перевірку введених даних:

- номер телефону має складатися рівно з 10 цифр;
- дата народження має бути у форматі `DD.MM.YYYY`.

# Команди асистента

| Команда | Опис |
|---|---|
| `hello` | Привітання з асистентом |
| `add name phone` | Додати новий контакт або телефон до існуючого контакту |
| `change name old_phone new_phone` | Змінити старий номер телефону на новий |
| `phone name` | Показати телефони контакту |
| `all` | Показати всі контакти |
| `add-birthday name birthday` | Додати день народження до контакту |
| `show-birthday name` | Показати день народження контакту |
| `birthdays` | Показати дні народження на найближчий тиждень |
| `close` | Завершити роботу |
| `exit` | Завершити роботу |
